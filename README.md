# Licipe|리시피

팀명: 앱 만들다 살찜

서비스명: Licipe

개발기간: 2021.07 ~ 2021.08(6주)

사용 스택: React, Redux, Material UI, HTML, CSS, JavaScript, AWS S3

개요: 음성인식 기술을 활용한 레시피 서비스. 레시피를 선택한 이후 영상 재생이나 레시피 단계 이동 등을 음성인식을 통해 제어 할 수 있습니다.
